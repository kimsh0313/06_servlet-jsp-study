Servlet이란?

1) 웹 서비스를 위한 Java 클래스
2) 자바를 활용하여 웹 개발을 하기 위해 필요한 기술
3) 사용자의 요청을 받아 처리하고 그에 해당하는 결과를 다시 사용자에게 응답하는 역할
4) 즉, Controller 측 Java 클래스라고 생각하면 됨
